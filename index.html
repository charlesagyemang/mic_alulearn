Fundamentals Of Software Engeneering

This course is about....

This module is an engineering branch related with growth of software product using well-defined scientific principles, methods and procedures. The outcome of software engineering is an efficient and reliable software product. Software project management has wider scope than software engineering process as it involves communication, pre and post-delivery support. The module Fundamentals of Software Engineering requires a good deal of reading. This reading will help you to build up an understanding that there is a lot more to software development than writing code!

  • Topic 1: Introduction to software engineering.
  • Topic 2: Software Engineering Processes.
  • Topic 3: Agile software development.
  • Topic 4: Requirement Engineering.
